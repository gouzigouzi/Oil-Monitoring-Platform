# 机油监测平台设计
机油监测平台设计过程分成三部分，对应前端、后端和算法的开发。

## [前端设计](fronted/vue-admin-template-master/)
使用 VScode 编辑器进行开发，基于开源框架 vue-admin-template 进行二次开发。

## [后端设计](backend/template/)
使用 IDEA IDE 进行开发，基于 Java 语言和 SprngBoot 框架开发，输入数据库为 Clickhouse，输出数据库为 Postgresql。

## [算法设计](algorithms/oil_pred/)
使用Pycharm IDE进行开发，基于Python3.8。
